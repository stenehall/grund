This is grund

My main reason for writing this was to better understand how to write compass modules and how css frameworks work.

To get it running you simply need to have compass installed and to the following.

$ cd your_repo_root/repo_name
$ gem install grund
$ compass create grund

Add require 'grund' to the newly created grund/config.rb

$ cd grund
$ compass init --require grund
